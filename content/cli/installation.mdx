# CLI Installation

## Requirements

- Node.js 18 or newer

## npm

```bash
npm install -g @globio/cli
```

## JSR

```bash
npx jsr add -g @globio/cli
```

## Verify the install

```bash
globio --version
```

## Log in

```bash
globio login
```

Or use a PAT non-interactively:

```bash
globio login --token glo_pat_xxxxx
```

## Confirm your session

```bash
globio whoami
```
