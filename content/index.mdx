# Globio Documentation

Globio is a game backend as a service built on
Cloudflare Workers. Give your game a complete
backend in minutes.

## What's included

| Service | Description |
|---|---|
| [GlobioId](/services/globio-id) | Authentication |
| [GlobalDoc](/services/global-doc) | Document database |
| [GlobalVault](/services/global-vault) | File storage |
| [GlobalPulse](/services/global-pulse) | Feature flags |
| [GlobalScope](/services/global-scope) | Analytics |
| [GlobalSync](/services/global-sync) | Real-time multiplayer |
| [GlobalSignal](/services/global-signal) | Notifications |
| [GlobalMart](/services/global-mart) | Game economy |
| [GlobalBrain](/services/global-brain) | AI integration |
| [GlobalCode](/services/global-code) | Edge functions |
